#Esta es una biblioteca personal

A continuación, les presento mi biblioteca persona en JS.

Para usarse:
1.Abre 'index.html' en el navegador
2. Abre la consola con F12
3. Ver los resultados de la biblioteca

Funcionalidades:
1- Valida si hay duplicados en agregar libros
2- Buscar libros por género
3- Prestar libros
4- Ver estadísticas de la biblioteca
